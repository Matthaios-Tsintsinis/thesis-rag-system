# My RAG Thesis Project
This repository contains the code for my thesis on RAG.

## How to Run (First Installation)
1. Put your PDF file(s) into the data/ folder.

2. Create a virtual environment:
	`python -m venv venv`

3. Activate the virtual environment (Windows):
	`venv\Scripts\activate`

4. Install the dependencies:
	`pip install -r requirements.txt`

5. Run the main script:
	`python main.py`

## How to Run (After Installation)
Whenever you return to the project, run these commands in your cmd:

	venv\Scripts\activate
	python naive_rag.py
